How to Update to Java 8

As you probably already know, you need to update to Java 8 in order to use the Wurst Client. This tutorial shows you exactly how to do that in the easiest and fastest way.

How to Update to Java 8 on Windows 8

  1. Download Java 8 and run the installer.
  2. Right click on the Windows logo and select Programs and Features.
  3. Find your old Java 6 / 7 installation and click Uninstall.
